## 녹화 체크리스트 (OBS Studio)

- [ ] **해상도**: 1920x1080 (landscape) 또는 1080x1920 (portrait)
- [ ] **프레임**: 30fps
- [ ] **비트레이트**: 6000 Kbps (AppSumo 권장)
- [ ] **오디오**: 마이크 -12 dB ~ -6 dB (피크 안 튀게)
- [ ] **씬 1**: Window Capture(Chrome) 100%
- [ ] **씬 2**: Window Capture(Chrome) 70% + 웹캠(우하단 25%)
- [ ] **트랜지션**: cut only (페이드 X — AppSumo는 빠른 페이스 선호)
- [ ] **인트로 자막**: 0:00에 `CheonMyeongDang | AppSumo Lifetime Deal` 1초 노출
- [ ] **최종 파일명**: `cheonmyeongdang_appsumo_demo_v1.mp4`
