## 1. 사용자 액션 9단계

### Step 1 — Solapi 가입
- 사이트: https://console.solapi.com
- 가입: 사업자 회원으로 가입 (개인사업자도 OK, 쿤스튜디오 사업자등록증 사용)
- 본인인증: 휴대폰 본인인증 필요 (사용자 직접 입력)
- 가입 후 좌측 메뉴 **"내 정보 → 사업자 정보"**에서 사업자등록증 사진 업로드

### Step 2 — 발신번호 사전등록 (선통신)
1. Solapi 콘솔 → **"발신번호 관리"**
2. **"발신번호 등록"** 클릭
3. 등록할 번호 (사용자 본인 휴대폰 또는 사업장 유선) 입력
4. 증빙 제출:
   - 사업자등록증 사본 (PDF/JPG)
   - **통신서비스 이용계약서** 또는 **최근 3개월 이내 통신요금 영수증**
   - 본인 명의가 아닌 경우 위임장 추가
5. 검토 1영업일, 통과 후 즉시 사용 가능
6. 등록 통과되면 `.secrets`에 다음 추가:
   ```
   SOLAPI_SENDER_PHONE=01012345678
   ```

### Step 3 — Solapi API Key 발급
1. Solapi 콘솔 → **"내 계정 → API Key 관리"**
2. **"API Key 생성"** 클릭 → 권한: **"메시지 발송 + 카카오 채널 관리"** 체크
3. `API Key`와 `API Secret` 즉시 복사 (Secret은 1회만 표시됨)
4. `.secrets`에 추가:
   ```
   SOLAPI_API_KEY=NCSXXXXXXXXXXXX
   SOLAPI_API_SECRET=XXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXX
   ```

### Step 4 — Solapi에서 카카오 채널 연동
1. Solapi 콘솔 → **"카카오 채널 → 채널 연동"**
2. **"채널 추가"** → 검색창에 `@cheonmyeongdang` 입력
3. 카카오톡 앱에서 **"채널 관리자 인증"** 알림 수락 (사용자 휴대폰)
4. 연동 완료 시 `pfId`(채널 토큰) 표시 → `.secrets`에 추가:
   ```
   KAKAO_PF_ID=908652
   ```
   (월렛 ID 908652가 곧 pfId)

### Step 5 — 카카오 비즈센터 접속 및 알림톡 메뉴 진입
1. https://center-pf.kakao.com 접속 → 카카오 계정 로그인
2. 좌측 채널 선택: **@cheonmyeongdang**
3. 좌측 메뉴 **"알림톡 → 템플릿 관리"** 진입
4. **"새 템플릿 등록"** 버튼 클릭

### Step 6 — 알림톡 템플릿 등록
**템플릿 원문**: `departments/cheonmyeongdang/kakao_template_daily_fortune.txt`

입력 필드 (그대로 복붙):

| 필드 | 값 |
|------|------|
| 템플릿명 | `천명당_매일운세_08시` |
| 카테고리 1차 | 회원관리 |
| 카테고리 2차 | 회원/구독 안내 |
| 메시지 유형 | 기본형 (BASIC) |
| 강조 표기 유형 | 선택안함 |

**템플릿 본문** (변수 3개 선언):

## 2. 알림톡 작성 규칙 (검수 통과 핵심)

### 분류
- **반드시 정보성**: 매일 정기 구독 알림은 정보성으로 등록
- **광고성 등록 금지**: "할인", "혜택", "이벤트", "쿠폰" 등 단어 1개라도 들어가면 반려

### 변수 형식 및 글자수
| 변수 | 최대 길이 | 비고 |
|------|----------|------|
| `#{name}` | 20자 | 회원 이름. 빈값 방지 위해 fallback "회원" 자동 적용 |
| `#{fortune}` | 160자 | 사주 풀이 요약. 1000byte 본문 한도 고려 |
| `#{score}` | 3자 | 1~100 정수 |

### 본문 규칙
- 고정 문구 **최소 1줄** 필수 (변수만으로 본문 구성 시 즉시 반려)
- 이모지 **최대 2개** 권장 (본 템플릿 본문은 0개, 친구톡 폴백만 사용)
- 1000byte (한글 약 333자) 한도

### 버튼 규칙
- 최대 5개, 권장 2~3개
- 외부 광고 링크 (쿠팡 파트너스 등 제휴) **절대 금지**
- URL은 자사 도메인 또는 Play Store/App Store만 안전

## 3. 검수 통과 팁

### 거절 사유 대응표

| 거절 사유 | 원인 | 수정 방법 |
|-----------|------|----------|
| 광고성 표현 포함 | "할인", "혜택", "특가", "지금 구매" 등 | 해당 단어 모두 제거, "안내", "도착" 등으로 치환 |
| 친구톡 vs 알림톡 혼동 | 광고성 본문을 알림톡으로 신청 | 광고성은 친구톡으로, 정기/회원 안내만 알림톡으로 |
| 변수 미사용 | 정형 메시지로 알림톡 의미 X | `#{name}` 1개 이상 변수 필수 (본 템플릿은 3개) |
| 외부 광고 링크 | 쿠팡/제휴 마케팅 URL | 자사 도메인만 사용 |
| 본문이 변수만으로 구성 | 고정 문구 0줄 | "[천명당] 오늘의 운세 도착" 등 고정 문구 1줄 이상 |
| 발송 대상 모호 | 누구에게 왜 보내는지 불명확 | "회원/구독자에게 매일 1회" 명시 |
| 이모지 과다 | 본문 이모지 3개 초과 | 0~2개로 축소 |

### 본 템플릿이 통과하는 이유
1. **분류 명확**: 회원관리 → 회원/구독 안내 (정보성)
2. **변수 3개 사용**: 회원별 개인화 메시지임을 증명
3. **고정 문구 풍부**: 본문 7줄 중 4줄이 고정
4. **광고 0**: "확인하세요" 같은 안내 문구만 존재
5. **버튼 자사 도메인**: vercel.app 자사 도메인 + 채널추가 버튼

### 첫 검수 거절을 줄이는 사전 점검
- [ ] 본문에 "할인/혜택/이벤트/쿠폰/특가/지금/구매" 단어 0개 확인
- [ ] 변수 최소 1개, 권장 3개 (이름/내용/숫자)
- [ ] 고정 문구 최소 1줄 (변수만으로 채우지 말 것)
- [ ] 외부 광고 링크 0개 (자사 도메인 + Play Store만)
- [ ] 이모지 0~2개
- [ ] 카테고리 "정보성" 또는 "회원관리" 선택
- [ ] 버튼 URL 정상 접속 확인
