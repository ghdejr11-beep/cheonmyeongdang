# 🏢 본사 (HQ) — CEO 직속 지침

> 이 폴더는 **대표이사 직속 공간**입니다. 부서 간 조정·전사 리포트·수익 랭킹을 관리합니다.

## 📂 파일 목록

- `scoreboard.md` — 월간 부서별 수익 랭킹 (매일 09:00 업데이트)
- `daily-report-template.md` — 일일 보고 표준 양식
- `weekly-report.md` — 주간 전사 종합 리포트

## 🕘 일일 보고 루틴

매일 **09:00 KST**, CEO는 다음 중 하나 명령:

1. `/daily-report` — 슬래시 커맨드 (모든 부서 일괄 보고)
2. 또는 텍스트: `"일일 보고"`

제가 받으면:
1. 7개 총괄팀장 subagent 병렬 호출
2. 각 부서가 `revenue.md`·`roadmap.md` 읽고 리포트 작성
3. 텔레그램 7개 메시지 발송 (부서당 1개)
4. `scoreboard.md` 업데이트
5. CEO에게 요약 제공

## 🏆 경쟁 규칙

- 수익 측정 단위: **KRW** (외화 수익은 환율 적용해서 원화 환산)
- 기준: **순수익** = 매출 - 도구비용(API·SaaS 등) - 외주비
- 집계 주기: 월간 (매월 1일 랭킹 발표)
- 꼴찌 부서: 다음 달 예산 10% 삭감 + 사유 분석 리포트
