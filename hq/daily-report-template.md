# 📋 일일 보고 표준 양식

> 매일 **09:00 KST**, 각 총괄팀장이 이 양식으로 보고.
> 최종 출력은 텔레그램 메시지로 전송.

---

## 📨 텔레그램 메시지 양식

```
🏢 [부서명] 일일 보고 | YYYY-MM-DD (요일) 09:00 KST

💰 수익
• 어제 매출: ₩XXX (USD 환산 $XXX)
• 어제 비용: ₩XXX
• 어제 순수익: ₩XXX
• 누적 이번달 순수익: ₩XXX
• 전일 대비: +XX% / -XX%

⚠️ 문제점 (어제~오늘)
• (이슈 1) — 영향도: 상/중/하
• (이슈 2) — 영향도: 상/중/하

🔧 개선 방안 (오늘 실행)
• (액션 1) — 담당: [팀장명]
• (액션 2) — 담당: [팀장명]

✅ 결과 / 계획
• 어제 완료: ...
• 오늘 예정: ...
• 이번 주 마일스톤: ...

📊 KPI (부서별 상이)
• (예: 유튜브 — 구독자 +XX / 조회수 XX)
• (예: 전자책 — 신규 출간 X권 / 판매 X권)
```

---

## 🔢 부서별 KPI 예시

### 🔮 천명당팀
- DAU / MAU
- 사주·관상·손금 각각 사용 횟수
- 유료 전환율
- 평균 체류 시간

### 📺 미디어팀
- YouTube 구독자 증감
- 조회수 (shorts / long)
- RPM (1천회 수익)
- SNS 팔로워 증감 (IG / TikTok / X)

### 🛡️ 보험다보여팀
- 심사 진행 상황 (금감원 D-X / 복지부 D-X)
- 앱 개발 진척도 (%)
- 연결된 설계사 수

### 📚 전자책팀
- 집필 중인 책 수
- Amazon KDP 출간 수 (누적)
- 권당 일일 판매 수
- 평균 로열티 수익 / 권

### 🎮 게임팀
- 프로토타입 진척도
- 플레이 테스트 피드백 수
- (활성화 후) DAU, ARPU

### 💼 세무팀
- 기획서 / 영업자료 진척도
- 파트너 미팅 수
- 계약 건수 (출시 후)

### 🗺️ 여행지도팀
- 지도 데이터 수집량 (POI 수)
- API 기획 완성도
- B2B 리드 수

## 📏 순수익 계산 공식

```
순수익(Net Revenue) = 매출(Revenue) - 변동비용(API·SaaS·외주)
```

**변동비용 예시:**
- fal.ai: 영상 생성 $X
- HeyGen: 아바타 $X
- OpenAI/Claude: 토큰 $X
- 광고비: $X
- 외주 디자이너: ₩X

환율: 매일 09:00 KST 기준 네이버 환율 사용. 1 USD = ₩1,400 기준.
