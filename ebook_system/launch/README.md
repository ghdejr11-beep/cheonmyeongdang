1억/100일 = 일 100만원. 현실적 조합:

|---|---|
| 랜딩 방문자 수 | Netlify Analytics 또는 GA |

- ROAS ≥ 2.0 → 광고 예산 일 5만원 → 10만원 → 20만원으로 증액

- 매출 0원 → 트래픽 0 문제인지 CVR 0 문제인지 분리 진단

## ⚡ 원칙

1. **매일 실행 > 완벽한 계획.** 이번 주에 10개 만드는 게 다음 주에 50개 계획 짜는 것보다 낫습니다.

3. **CVR 1% 이하면 트래픽 늘리지 마라.** 랜딩을 먼저 고쳐야 함.
4. **후기 1개가 광고 10개보다 강력.** Day 6 부터 매일 후기 수집.
5. **한 달 후에 안 되면 재평가.** 30일 후 매출 500만원 미만이면 주제/카피/타깃 전부 검토.
