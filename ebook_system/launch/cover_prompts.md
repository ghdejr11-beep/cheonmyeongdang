# 🎨 책 표지 AI 생성 프롬프트 5개

Gumroad/랜딩페이지에 쓸 책 표지를 **Midjourney / Leonardo AI / Ideogram** 등으로 5분 만에 만드는 프롬프트.

**원칙**: 책 표지는 **3초 안에 전하는 약속**이 전부. 디자인 완성도보다 **인지도 / 클릭률** 이 중요.

---

## 🚀 빠른 사용법

### 방법 1: Leonardo AI (무료 150크레딧/일, 추천)
1. https://leonardo.ai 가입
2. **Image Generation** → 모델: **Phoenix 1.0** 또는 **Leonardo Kino XL**
3. 아래 프롬프트 중 하나 복붙
4. 비율: **2:3 (책 표지)** 또는 **1024x1536**
5. 생성 → 마음에 드는 것 다운로드

### 방법 2: Midjourney (월 $10)
1. Discord 에서 `/imagine` 입력
2. 프롬프트 뒤에 `--ar 2:3 --v 6` 추가
3. 4개 생성 → 좋은 것 `U1~U4` 로 upscale

### 방법 3: Ideogram (무료, 텍스트 정확함)
1. https://ideogram.ai 가입
2. 프롬프트 입력 (텍스트 포함된 표지에 특히 좋음)
3. 비율: 2:3

## 🧪 A/B 테스트 팁

**3개 표지 동시 제작** → Gumroad/인스타 광고에서 2주 테스트 → 클릭률 1위 선택.

보통은:
- 어두운 배경 > 밝은 배경 (CTR +15%)
- 숫자 포함 > 숫자 없음 (CTR +20%)
- 얼굴 포함 > 얼굴 없음 (CTR +10%)

## 📐 완성 후 후처리 (선택)

**Canva** (무료) 에서:
1. AI 생성 표지 업로드
2. 텍스트가 흐리면 Canva 에서 한국어 텍스트 오버레이 (Pretendard 폰트 추천)
3. 가격 뱃지 추가 (예: "50% OFF")
4. 출력: 1600×2400 PNG

---

## 🆓 완전 무료로 만드는 방법

표지를 이미지 생성 없이 Canva 템플릿만으로:
1. https://www.canva.com → "book cover" 검색
2. "business book" 필터
3. 맘에 드는 템플릿 선택
4. 제목/색상만 변경
5. 다운로드

10분이면 끝. AI 없어도 품질 충분.

## 💡 팁: 표지 예산 제로로 해결하는 경우

Gumroad 는 기본 표지 기능이 있어서, 극단적으로는:
- **단색 배경 + 큰 제목** 만으로도 충분
- 오히려 심플한 게 베스트셀러 (e.g., "The E-Myth Revisited" 원서 표지)

핵심은 **표지에 시간 쓰지 말고, 콘텐츠/트래픽에 쓰는 것**.
처음엔 AI 로 2~3개 후보 뽑고, 주말에 30분만 투자해서 결정하세요.
