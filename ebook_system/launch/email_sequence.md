# 📧 이메일 시퀀스 5개 — 무료 샘플 → 본책 전환

**용도**: 랜딩 페이지의 "무료 샘플 받기" 폼에서 이메일 구독 → 5일간 자동 발송 → 본책 판매 전환

**플랫폼**: Stibee (한국, 무료 500명까지) 또는 ConvertKit

**전환율 기대치**: 이메일 100명 구독 → LITE 5~10명 구매 + STANDARD 2~3명 구매

## ⚙️ Stibee 설정 방법 (3분)

1. https://stibee.com 가입 → 무료 플랜
2. **주소록 생성** → 이름: `ai-ebook-subscribers`
3. **자동화** 메뉴 → **새 자동화** → **구독 시**
4. 위 5개 이메일을 각각 등록:
   - Email 1: 구독 즉시
   - Email 2: 구독 후 1일
   - Email 3: 구독 후 3일
   - Email 4: 구독 후 5일
   - Email 5: 구독 후 7일 (오후 8시)
5. `[구독자 이름]`, `[TEASER_PDF_URL]`, `[STANDARD_URL]` 부분을 본인 값으로 치환
6. **구독 폼** 생성 → 임베드 코드 복사 → `landing.html` 에 삽입

## 📊 전환율 기대치 (현실적)

| 퍼널 단계 | 전환율 | 수치 예시 |
|---|---|---|
| 랜딩 방문자 | 100% | 1,000명 |
| → 이메일 구독 | 10~15% | 100~150명 |
| → Email 1 열람 | 50% | 50~75명 |
| → Email 5 까지 읽음 | 20% | 20~30명 |
| → LITE 구매 | 5% | 5~7명 (약 10만원) |
| → STANDARD 구매 | 3% | 3~4명 (약 30만원) |
| → PREMIUM 업셀 | 1% | 1명 (약 30만원) |
| **총 매출 (1,000 방문자 기준)** | | **약 70~100만원** |

→ 1억 도달하려면 누적 방문자 100,000~150,000명 필요
→ 일 1,500명 × 100일 = 150,000명 (가능)
