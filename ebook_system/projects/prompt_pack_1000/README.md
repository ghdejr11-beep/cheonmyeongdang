## 🎯 왜 이 상품인가

**프롬프트 팩은 100일 1억의 숨은 열쇠**입니다.

1. **환불율 극히 낮음**: 복붙해서 바로 쓰는 자료는 "안 맞는다" 가 적용 안 됨
2. **AOV 상승**: AI 부업 책(99,000) + 프롬프트 팩(29,900) 번들 = 객단가 30% 증가
3. **빠른 제작**: Claude 가 30분 만에 생성. 저자 직접 집필 불필요
4. **평생 콘텐츠**: 프롬프트 원리는 모델 바뀌어도 유효
5. **크로스 오더 게이트웨이**: 저가 상품이 STANDARD/PREMIUM 업셀 유도

## 🚀 빠른 시작 (3단계)

### Step 1: 환경변수 확인

```powershell
echo $env:ANTHROPIC_API_KEY
```

→ 본인 Claude API 키가 출력돼야 함. 안 되면 세팅부터.

### Step 2: 1,000개 프롬프트 생성 (15~30분)

```powershell
cd C:\Users\쿤\Desktop\cheonmyeongdang\ebook_system\projects\prompt_pack_1000
python generate_prompts.py
```

진행 상황이 카테고리별로 화면에 표시됩니다:
```
▶ [email] 📧 이메일 마케팅 & 세일즈 — 100개 생성 중...
    ✓ 100개 생성 완료
▶ [copywriting] ✍️ 카피라이팅 & 광고 — 100개 생성 중...
    ...
```

중간에 끊겨도 `output/prompts.json` 에 저장되므로 다시 실행하면 이어서 진행.

### Step 3: PDF 변환 (1~2분)

```powershell

```

→ `output/prompt_pack_1000.pdf` 생성 (약 3~5MB, 200~300 페이지)

## ❓ FAQ

**Q. 1,000개를 전부 Claude API 로 만들면 품질이 괜찮은가요?**
A. Haiku 4.5 도 구조적인 프롬프트 생성은 아주 잘합니다. 단, 더 높은 품질을 원하시면 `config.py` 에서 `MODEL = "claude-sonnet-4-6"` 으로 변경 (비용은 3배 증가).

**Q. 생성된 프롬프트 중 일부가 마음에 안 들면?**
A. `output/prompts.json` 을 직접 열어서 편집 가능. 텍스트 파일이라 쉽게 수정됩니다. 그 후 `python make_prompt_pdf.py` 로 재빌드.

**Q. 카테고리를 추가하거나 바꾸고 싶으면?**
A. `config.py` 의 `CATEGORIES` 리스트를 수정. 각 카테고리의 `count` 도 조정 가능 (예: 200개로).

**Q. 생성된 JSON 을 한번에 직접 편집하고 싶은데 너무 큼.**
A. VSCode 또는 노션에 붙여넣으면 구조화되어 보입니다. 또는 `python -c "import json; [print(p['title']) for p in json.load(open('output/prompts.json'))['prompts']]"` 로 목차만 확인.

**Q. 저작권은 어떻게 되나요?**
A. Claude API 로 생성된 텍스트는 사용자(=본인) 에게 저작권이 있습니다. 단, 상업적 판매는 Anthropic 이용약관 준수 필요. 일반적인 프롬프트 판매는 허용됨.
