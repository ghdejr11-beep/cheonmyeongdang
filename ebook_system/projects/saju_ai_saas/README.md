## ⚠️ 주의사항

### 1. 법적 리스크 회피
- 점술 서비스는 한국에서 **합법** 이지만, **과장 광고** 금지
- "운명을 100% 맞춥니다" 같은 문구 ❌
- "재미와 참고용" 이라는 문구 필수
- 의료·법률·재무 결정에 대한 직접 조언 금지

### 2. 데이터 보호
- 생년월일 = 개인정보. SQLite 에만 저장 (외부 전송 금지)
- HTTPS 필수 (Render 기본 제공)
- 세션 쿠키 httponly + secure

### 3. Claude API 비용 관리
- 프롬프트 캐싱 적용 (사주 컨텍스트 재사용)
- 답변 max_tokens=1024 제한
- 대화 히스토리 최근 10턴만 유지
- 예상 비용: 사용자 1명당 월 $0.3~0.5 (Haiku)

## ❓ FAQ

**Q. 사주 계산이 정확한가요?**
A. MVP 는 85% 정확도. 절기 경계일 출생은 ±1일 오차 가능. v1.5 에서 KASI API 연동 예정.

**Q. Claude 가 진짜 사주를 풀 수 있나요?**
A. Claude 는 명리학 기초 지식 있음 (학술 논문, 위키, 도서 학습). 시스템 프롬프트로 톤·구조·원칙을 통제하면 인간 역술인과 유사한 품질 가능.

**Q. 사용자가 "이거 AI 냐?" 라고 물으면?**
A. **투명하게 답**. "Claude AI 기반 사주 상담 서비스입니다" 라고 명시. 숨기면 신뢰 잃음.

**Q. 환불 요청 오면?**
A. 7일 안에 무조건 승인. 환불율 10% 미만이면 정상. Gumroad 가 자동 처리.

**Q. 경쟁사가 카피하면?**
A. 경쟁사도 Claude 씁니다. 차별화는 **천명당 브랜드 + 한국 문화 이해 + SEO 자산** 입니다.
