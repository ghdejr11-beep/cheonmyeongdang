▶ 결제경로 (Payment Flow)

[1단계] 사이트 방문
  URL: https://cheonmyeongdang.com (또는 cheonmyeongdang.vercel.app)

[2단계] 사주 입력
  - 메인 페이지에서 생년월일·시(時)·성별 입력
  - 무료 미리보기 결과 1차 확인

[3단계] 프리미엄 상품 선택
  - "프리미엄 보기 (₩9,900)" 또는 "월회원 (₩9,900/월)" 버튼 클릭
  - 단건결제 / 정기결제 두 가지 노출

[4단계] 결제창 호출 (PortOne V2 SDK)
  - JavaScript: PortOne.requestPayment({ channelKey: 'channel-key-kakaopay', payMethod: 'EASY_PAY', easyPay: { provider: 'KAKAOPAY' } })
  - 카카오페이 결제창 팝업 → 카카오 인증 (앱/QR/SMS)
  - 결제 완료 후 redirect

[5단계] 결제 검증·후처리
  - api/confirm-payment.js (Vercel Serverless)
  - PortOne API로 결제 검증 → 성공 시 success.html
  - api/send-receipt.js로 영수증 메일 자동 발송

[6단계] 환불 처리 (요청 시)
  - 24시간 이내 환불 정책 (사이트 푸터·체크아웃 페이지에 명시)
  - api/refund-confirm.js → PortOne 환불 API 호출 → 환불 영수증 자동 발송

▶ 결제 형태

  단건결제 (One-time):
    - ₩9,800 사주 프리미엄 PDF
    - ₩7,500 영문 단건 (글로벌)

  정기결제 (Subscription):
    - ₩9,900/월 무제한 사주·운세·꿈해몽
    - ₩29,000/월 프리미엄 (전 컨텐츠 + B2B API)

▶ PG 채널 상태

  - PortOne V2 SDK 통합 완료
  - 1순위: 한국결제네트웍스(KCN) 라이브 심사 중
  - 2순위: 갤럭시아머니트리(빌게이트) 백업 채널 신청 완료
  - 카카오페이 채널: 본 심사 통과 즉시 정기결제·단건결제 모두 활성화 예정
