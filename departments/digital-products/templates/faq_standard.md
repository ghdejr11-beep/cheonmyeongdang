> 이 파일은 모든 listing(`prompts/listing_*.md`, Gumroad/크티/Etsy 상세) 하단에 삽입되는 **공통 FAQ 블록**입니다.
> 출처: `pricing_tiers.json` → `faq_template` (7문항)
> 업데이트 기준일: 2026-04-25

## 사용 방법

각 listing 마크다운에 아래 토큰을 한 번만 넣으면, 자동 빌드 시 이 블록 전체로 치환됩니다.

```text
{{FAQ_BLOCK}}
```

빌드 스크립트는 토큰을 정확히 일치 검색 후 아래 "## 자주 묻는 질문" 섹션 전체로 교체합니다.
공백/줄바꿈 차이 시 치환이 안 되니 토큰을 그대로 복붙하세요.

## 자주 묻는 질문

### Q1. 환불 가능한가요?
**A.** 네. 구매 후 **30일 이내 무조건 환불**됩니다. 사유 불문, 한 줄 메일이면 처리됩니다.
- 환불 메일: support@kunstudio.co
- 평균 처리 시간: 영업일 기준 24시간 이내

### Q2. 업데이트는 어떻게 받나요?
**A.** Gumroad 시스템을 통해 **자동 알림 + 자동 재다운로드**가 제공됩니다.
- 새 버전이 올라가면 구매자 전원에게 이메일이 발송됩니다.
- 추가 비용 없이 평생(또는 라이선스 기간) 받을 수 있습니다.
- Pro/Bundle 티어는 **월 1회 업데이트가 12개월간 보장**됩니다.

### Q3. 한국어 가이드가 있나요?
**A.** 네. **한국어 + 영어 가이드 모두 포함**되어 있습니다.
- 한국어 PDF: 사용법, 카테고리별 예시, 응용 팁
- 영어 PDF: 동일 내용의 영문판 (해외 판매용)
- ChatGPT / Claude / Gemini 어떤 모델에서도 동일하게 동작합니다.

### Q4. 상업적으로 사용해도 되나요?
**A.** 티어별로 다릅니다.
- **Lite**: 개인 학습/사용만 가능 (상업 이용 ❌)
- **Pro**: 본인 사업장/SNS/유튜브 등 **상업적 활용 가능** ✅
- **Bundle**: 상업적 활용 + **재판매 일부 권한** (조건은 라이선스 문서 참고)
- 어떤 티어든 **프롬프트를 그대로 재판매하는 행위**는 금지됩니다.

### Q5. 결제는 어떻게 하나요?
**A.** 다양한 방법이 지원됩니다.
- **신용카드 / 체크카드** (Visa, Master, Amex)
- **PayPal** (해외 구매자용)
- **네이버페이 / 카카오페이** (크티 구매 시)
- **계좌이체** (대량 구매 / 법인 구매 문의)
- 영수증/세금계산서 발급 가능 (사업자번호 알려주시면 자동 발행)

### Q6. 구매 후 어디서 다운로드하나요?
**A.** 두 가지 경로로 받을 수 있습니다.
1. **이메일 링크** — 결제 즉시 발송되는 메일의 다운로드 버튼
2. **마이페이지** — Gumroad 로그인 후 "Library"에서 언제든 재다운로드 가능
- 다운로드 횟수 제한 없음
- 새 기기에서도 같은 이메일로 로그인하면 즉시 받을 수 있습니다.

### Q7. 기술 지원은요?
**A.** 티어별 지원 수준이 다릅니다.
- **Lite**: 이메일 응답 (영업일 48시간 이내)
- **Pro**: 이메일 응답 (**24시간 이내 보장**) + Notion FAQ 접근
- **Bundle**: 24시간 이내 응답 + **1:1 30분 코칭 1회 제공**
- 지원 채널: support@kunstudio.co | 카카오 비즈채널 @kunstudio

## 토큰 통합 가이드 (개발자용)

| 토큰 | 치환 결과 |
|---|---|
| `{{FAQ_BLOCK}}` | 위 "## 자주 묻는 질문" 섹션 전체 (Q1~Q7) |
| `{{FAQ_SHORT}}` | Q1, Q2, Q4, Q7 만 (4문항 축약본) |
| `{{SELLER_INFO}}` | 위 "## 판매자 정보" 섹션 |

빌드 스크립트 예시 (Python):

```python
from pathlib import Path

faq = Path("departments/digital-products/templates/faq_standard.md").read_text(encoding="utf-8")
# "## 자주 묻는 질문" 부터 다음 "---" 직전까지를 추출
block = faq.split("## 자주 묻는 질문", 1)[1].split("---", 1)[0]
faq_block = "## 자주 묻는 질문\n" + block.strip()

listing = Path("departments/digital-products/prompts/listing_sns.md").read_text(encoding="utf-8")
listing = listing.replace("{{FAQ_BLOCK}}", faq_block)
```
