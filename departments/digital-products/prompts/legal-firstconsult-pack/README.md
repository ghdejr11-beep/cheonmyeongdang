- "임대차 보증금 못 받았다는 카톡 문의 → 1차 답변 + 변호사 상담 권유" (3분 → 30초)
- "유언 작성 문의 → 자필증서/공정증서 차이 안내 카톡" (즉시)
- "퇴직금 못 받았다 카톡 → 노동청 진정 1차 안내" (1분)
- "음주운전 1차 처분 안내 카톡 → 변호사 상담 연결" (1분)
