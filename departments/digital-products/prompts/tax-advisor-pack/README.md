# 세무 답변 프롬프트팩 100선

세무사·세무사무소 직원·기장 대행·1인 사업자 회계 담당자를 위한 한국어 AI 프롬프트 100개 모음.
종합소득세, 부가세, 사업자등록, 환급, 고객 불만 응대까지 — 세무사법 가이드라인 안에서 1차 답변을 자동화하는 실전 프롬프트만 모았습니다.

## 누가 써야 하나요?
- 1인 세무사 + 직원 1~2명 사무소 (월 100~300건 카톡 문의)
- 기장 대행 5인 미만 사무소
- 삼쩜삼·세이브어택·SSEM 같은 세무 SaaS 운영자
- 회계·재무 담당 1인 사업자

## 활용 예시
- "고객 카톡 문의 → 종합소득세 1차 답변 자동 생성" (3분 → 30초)
- "신규 사업자 등록 안내 카톡 5종" (한 번 작성 후 영구 사용)
- "삼쩜삼이 환급 못 받았다는 고객 불만 응대" (감정+정보 균형)
- "부가세 신고 마감 전 D-7 일별 알림 카톡" (자동 발송 텍스트)

## 카테고리 (5종 × 20개 = 100)
1. 종합소득세 신고 안내
2. 부가세 신고 답변
3. 사업자 등록 안내
4. 환급 절차 설명
5. 고객 불만 응대

## 호환
- ChatGPT (GPT-4 / GPT-4o / GPT-5)
- Claude (Sonnet 4.6 / Opus 4.7)
- Gemini (1.5 Pro / 2.0)
- 네이버 클로바X / 뤼튼 / Wrtn

## 가격
- Lite ₩12,900 (단일 카테고리 100개)
- ★ Pro ₩39,000 (5개 카테고리 + 노션 템플릿 5종 + 영상 가이드)
- Bundle ₩99,000 (전체 + 12개월 업데이트 + 1:1 30분 코칭)

## 법적 고지
본 프롬프트팩은 **세무사법 제20조(직무의 정의)에 따른 세무 대리 업무가 아닌, 세무 관련 정보 안내 보조 도구**입니다. 본 팩의 프롬프트 결과물은 일반적인 안내 수준이며, **개별 고객의 구체적인 세무 신고·세액 계산·세무 자문은 반드시 자격을 갖춘 세무사가 수행**해야 합니다. 세무사 자격 없는 자가 본 팩을 사용해 세무 대리 업무를 영위할 경우 세무사법 제22조(무자격자의 세무 대리 금지) 위반에 해당할 수 있습니다.
