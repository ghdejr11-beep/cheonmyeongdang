## 핵심 기능 (MVP)

### 탭 1: 💰 내 환급금
- 홈택스 연동 (공동인증서/간편인증)
- 5년치 종합소득세 환급금 자동 조회
- 원클릭 신고 (국세청 API)
- **수수료 0%** 대문짝만하게 표시

### 탭 2: 📊 절세 코칭
- 내 소득/지출 분석
- "이번 달 경비 처리하면 세금 OO만원 절감" 실시간 알림
- 직종별 절세 가이드 (배달라이더/유튜버/프리랜서/사업자)
- 부양가족 공제, 노란우산공제 등 자동 체크

### 탭 3: 🏛️ 정부 지원금
- 내 조건(나이/소득/지역/업종) 입력
- 매칭 가능한 정부 지원사업 자동 추천
- 신청 가능 여부 + 신청 방법 + 마감일 알림
- 데이터 소스: 정부24 API, 복지로 API, 소상공인시장진흥공단

### 탭 4: 👨‍💼 세무사 상담
- AI 챗봇이 1차 상담 (90% 해결)
- 복잡한 건 → "찾아줘 세무사" 연결
- 세무사 프로필 (지역/전문분야/평점)

### 탭 5: 📋 마이페이지
- 사업자 정보 관리
- 세금 납부 일정 캘린더
- 프리미엄 구독 관리

## 기술 스택
- Frontend: React Native 또는 HTML5 (Capacitor)
- Backend: Supabase 또는 Firebase
- AI: Claude API (세무 상담)
- API 연동: 홈택스 API, 정부24 API, 복지로 API
- 결제: 토스페이먼츠 (구독)
- 광고: AdMob
