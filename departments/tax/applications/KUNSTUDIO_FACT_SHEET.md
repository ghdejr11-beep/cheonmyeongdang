## 2. 제품 라인업 (5개 + α)

### 2.1 천명당 (cheonmyeongdang.vercel.app) — AI 사주·관상·궁합 SaaS
- **무엇**: 한국 전통 사주(四柱)·손금·꿈해몽·궁합·신년운세 AI 진단
- **AI 기술**: Claude API (Sonnet 4.6) + Pollinations Flux (시각화) + 자체 사주 데이터베이스 350+
- **타깃**: 글로벌 영문 사용자 (한국 문화 관심 외국인 + K-pop 팬)
- **수익 모델**: PayPal Smart Buttons (무료 진단 + 프리미엄 ₩2,900~₩29,900)
- **상태**: 라이브 (Vercel Hobby + AdSense 심사 중)
- **차별화**: 한국 명리학 정통성 + AI 자동화 + 영문 글로벌 진출

### 2.2 KORLENS (korlens.app) — AI 한국 여행 가이드
- **무엇**: 한국 여행자용 4개국어 (한/영/일/중) 가이드 앱 + AI "현지인 픽"
- **AI 기술**: Supabase + Google OAuth + AI 추천 알고리즘
- **타깃**: 한국 인바운드 여행자 (3,000만 명/년 시장)
- **수익 모델**: 어필리에이트 (호텔·체험·식당) + 프리미엄 가이드 PDF
- **상태**: Play Console 비공개 테스트 중

### 2.3 세금N혜택 (tax-n-benefit-api.vercel.app) — AI 종합소득세 환급 서비스
- **무엇**: 종소세 신고 자동화 + 환급액 계산 + 세무사 연결
- **AI 기술**: Claude API + 국세청 API 연동
- **타깃**: 5월 종소세 시즌 자영업자/프리랜서 (1,000만 명+)
- **수익 모델**: 환급액 5~10% 수수료 (B2C)
- **상태**: 라이브 + 빌게이트 PG 통합

### 2.4 K-Wisdom YouTube 채널 + Sori Atlas — AI 영상 자동 제작
- **무엇**: 한국 문화 글로벌 영문 콘텐츠 일일 자동 제작·업로드
- **AI 기술**: Claude API (스크립트) + Pollinations Flux (썸네일) + Suno (음악) + ffmpeg (영상)
