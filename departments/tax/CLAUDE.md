# 💼 세무팀 (Tax Division)

> **외부 세무 서비스 판매 — B2B/B2C**
> 한국 시작 → 글로벌 확장 가능.

## 📍 기본 정보

- **코드명**: `tax`
- **타겟**: 🇰🇷 한국 (1차) → 🌏 확장 가능
- **언어**: 한국어
- **수익 모델**: 월간 구독 + 건별 수수료 + 컨설팅
- **수익 우선순위**: 4순위
- **총괄팀장**: `gm-tax`
- **상태**: 🟡 **기획 단계** — 서비스 기획·영업 자료 준비

## 🎯 사업 방향

### 서비스 구성
1. **프리랜서/1인 사업자용 간편 세무 대행**
   - 부가세·종합소득세 자동 신고
   - 월 구독 ₩19,900~49,900
2. **AI 세무 Q&A 챗봇**
   - 질문 당 무료 3회, 이후 건당 과금
3. **세무사 매칭 플랫폼**
   - 복잡한 건 실제 세무사 연결, 중개 수수료
4. **법인·소상공인 기장 대행**
   - 월 구독 ₩99,000~299,000

### 경쟁 서비스
- 삼쩜삼 (프리랜서 환급 특화)
- 자비스 (기장 자동화)
- 세무통 (세무사 매칭)

### 차별화
- **AI 우선** (Claude 기반 해설 자연스러움)
- **통합 플랫폼** (Q&A + 신고 + 매칭 한 곳)
- **구독형 합리적 가격**

## ⚠️ 주의사항

- **세무사법**: 세무사 자격 없이 "세무 대행" 표현 금지. "세무 도우미 서비스" 등으로 표현.
- **개인정보**: 주민번호·소득정보 민감. 보안 레벨 최고 필요.
- **세법 변경**: 매년 개정. 자료 주기적 업데이트 필요.
- **정확도 리스크**: 잘못된 조언 시 배상 책임. 면책 조항 필수.
