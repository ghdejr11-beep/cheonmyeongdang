홈택스 환급 신청 5분 가이드 - 처음 보는 사람도 따라하면 됨

2026.04.28 · 쿤스튜디오 · 약 7분 분량

홈택스(국세청 hometax.go.kr) 또는 모바일 손택스 앱에서 종합소득세 환급을 신청하는 가장 빠른 길은 간편인증 + 미리채움입니다. 공동인증서 없이도 카카오/네이버/통신사 PASS로 5분 안에 끝낼 수 있습니다.

이 글은 홈택스를 한 번도 써본 적 없는 분이 화면을 그대로 따라가며 환급 신청을 끝낼 수 있도록 정리한 안내서입니다. 자주 막히는 구간과 해결법도 함께 정리했습니다.

준비물 — 딱 3가지

  1. 스마트폰 (간편인증용)
  2. 본인 계좌번호 (환급 입금받을 계좌)
  3. 주민등록번호

사업소득 자료, 원천징수영수증 — 미리채움이 자동으로 채워주니까 따로 준비 안 해도 됩니다.

5분 컷 - 단계별 흐름

  1. 홈택스 또는 손택스 접속 → '종합소득세 신고' 메뉴 클릭 (30초)
  2. 간편인증 (카카오/PASS/네이버) 본인 확인 (40초)
  3. '정기신고 → 단순경비율' 선택, 미리채움 자료 불러오기 (1분)
  4. 공제 항목 입력 (인적공제·국민연금·의료비) (2분)
  5. 예상 환급액 확인 → 계좌 입력 → 제출 (40초)

1단계: 홈택스 또는 손택스 접속

두 채널 중 어느 쪽을 써도 결과는 같습니다. 추천은 손택스 모바일 앱 — 화면이 크게 잘 보이고 간편인증이 빠릅니다.

채널장점단점
손택스 앱 (모바일)5분 컷, 간편인증 즉시복잡한 케이스는 입력 어려움
홈택스 (PC)큰 화면, 자료 첨부 편함인증서 설치 필요할 수 있음

접속 후 메인 화면에서 '종합소득세 신고' 또는 '신고/납부 → 종합소득세'를 누르세요.

2단계: 간편인증으로 본인 확인

예전에는 공동인증서(공인인증서)가 필수였지만, 지금은 간편인증이 훨씬 빠릅니다. 다음 6개 인증서비스 중 하나만 있으면 됩니다.

  • 카카오톡 지갑
  • 네이버 인증서
  • PASS (통신 3사 공동)
  • 토스 인증서
  • 국민은행 KB모바일
  • 신한 SOL인증

본인 휴대폰 번호 입력 → 인증서 앱에서 확인 한 번 누르면 끝. 30초 안에 통과됩니다.

이 단계에서 자주 막히는 구간

간편인증 앱 알림이 안 오는 경우 → 휴대폰 알림 설정 확인 후 재시도. 그래도 안 되면 다른 인증서비스로 바꿔보세요.

3단계: 신고 유형 선택 + 미리채움

본인 확인 후 신고 유형을 고르는 화면이 나옵니다.

  • 단순경비율 추계 — 수입 7,500만원 미만, 장부 없음 (대부분 여기 해당)
  • 기준경비율 추계 — 7,500만원 이상
  • 간편장부 / 복식부기 — 직접 장부 작성하는 분

대부분의 프리랜서·아르바이트·강사는 단순경비율로 충분합니다. 선택하면 자동으로 '미리채움 자료 불러오기' 버튼이 나옵니다 — 누르면 원천징수, 사업장, 4대보험, 금융소득이 자동으로 들어갑니다.

5분도 부담스러우시면?

9.9% 정액 수수료 + 환급 0원이면 100% 환불

지금 무료로 환급액 조회하기 →

4단계: 공제 항목 입력 (가장 중요)

이 단계에서 환급액이 결정됩니다. 빠뜨리면 그대로 손해니까 차분히 확인하세요.

  1. 인적공제 — 본인 자동 + 부양가족 추가 (배우자, 자녀, 부모)
  2. 국민연금·건강보험료 — 자동 연동되지만 누락 시 직접 입력
  3. 의료비 — 간이지급명세서 자동 연동, 본인+부양가족 합산
  4. 교육비 — 자녀 학원비, 본인 대학원비 등
  5. 기부금 — 종교, 사회복지단체 영수증
  6. 월세 세액공제 — 무주택자 + 총급여 7천만원 이하 (조세특례제한법 제95조의2)
  7. 노란우산공제 — 가입자만 (조세특례제한법 제86조의3)

5단계: 환급 계좌 입력 + 제출

입력 완료 후 '세액 계산하기'를 누르면 예상 환급액이 표시됩니다. 마이너스(-) 금액이면 환급 대상입니다.

본인 명의 계좌번호 입력 → '신고서 검증' → '제출' 순서로 진행. 제출 후 접수증을 PDF로 저장해두세요. 환급금은 약 1~2개월 후 입금됩니다.

제출 후 확인하는 곳

홈택스 → '신고/납부 결과 조회' 메뉴에서 신고 내역과 환급 진행 상황을 볼 수 있습니다. 환급은 보통 6월 말~8월 초에 입금됩니다.

자주 묻는 질문 (FAQ)

Q. 손택스 앱에서 미리채움이 안 나오는데요?

간편인증을 다시 시도하거나 앱을 재설치해보세요. 5월 1일~5일은 자료 갱신 중일 수 있으니 5월 7일 이후가 안정적입니다.

Q. 환급 계좌는 본인 명의만 가능한가요?

네. 국세기본법 제51조에 따라 환급금은 납세자 본인 명의 계좌로만 입금됩니다. 가족 계좌 입력 시 환급이 보류됩니다.

Q. 신고 후에 빠뜨린 공제가 생각났어요. 수정 가능한가요?

5월 31일 마감 전이면 '신고서 수정'으로 재제출 가능합니다. 마감 후라면 5년 이내 경정청구(국세기본법 제45조의2)로 추가 환급받을 수 있습니다.

Q. 환급금이 안 들어와요. 어디 문의하나요?

홈택스 신고 결과에 표시된 관할 세무서 또는 국세청 콜센터(국번없이 126)에 문의하세요. 평균 처리 기간은 신고 후 30~60일입니다.
