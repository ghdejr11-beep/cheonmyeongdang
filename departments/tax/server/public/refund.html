CORE PROMISE

환급액 0원이면 100% 자동 환불합니다.

별도 신청 절차 없이, 환급액 확정일로부터 7영업일 이내 결제하신 카드/계좌로 자동 반환됩니다.

환불 등급 (4단계)

신청 시점과 처리 단계에 따라 환불 비율이 달라집니다.

100%

① 신청 후 24시간 이내 취소

이유 불문, 24시간 이내 마음이 바뀌면 수수료 전액을 환불합니다. CODEF API 호출 전이면 즉시 환불됩니다.

100% 자동

② 환급액 0원 확정

국세청이 환급액을 0원으로 확정한 경우 별도 신청 없이 자동 환불됩니다 (확정일+7영업일).

50%

③ 처리 시작 후 취소

CODEF API 호출 등 실제 처리가 시작된 이후 회원이 자발적으로 취소하는 경우 처리 비용을 공제하고 50%를 환불합니다.

불가

④ 환급금 입금 완료 후

국세청 환급금이 회원 계좌로 정상 입금된 이후에는 단순 변심에 의한 환불은 불가합니다. (회사 귀책 사유는 별도)

회사 귀책(시스템 오류·환급 누락 등) 100% 회사의 고의·과실로 환급이 누락·오처리된 경우
처리 시작 후 회원 취소 50% CODEF API 호출 후 회원의 자발적 취소

회원의 허위 자료 제출로 인한 환급 거절 불가 처리 비용 발생, 환불 제한 (이용약관 §8)

2.환불 신청 방법

  1. 이메일 신청. ghdejr11@gmail.com 으로 다음 정보를 보내주세요.
    • 회원 이름 / 가입 이메일
    • 신청 일자 / 주문번호 (있을 경우)
    • 환불 사유
    • 환불 받을 계좌 정보 (결제와 동일한 카드 환불이 원칙이며, 부득이한 경우 계좌 환불)
  2. 접수 확인. 영업일 기준 1일 이내 접수 확인 메일을 발송합니다.
  3. 심사·처리. 환불 사유 확인 후 본 정책에 따라 환불 비율을 산정하고, 영업일 7일 이내 처리합니다.
  4. 완료 통지. 환불 완료 후 처리 결과를 메일로 안내합니다.
자동 환불 대상. "환급액 0원 확정" 케이스는 별도 신청 없이 자동 처리됩니다. 별도의 메일 발송 없이 결제 수단으로 즉시 반환됩니다.

3.환불 처리 기간

  • 카드 결제 환불 — 회사 처리는 영업일 3일 이내 완료, 카드사 정산까지 통상 5~7영업일 추가 소요
  • 계좌 환불 — 회사 처리 후 영업일 1~2일 이내 입금
  • 환급액 0원 자동 환불 — 확정일로부터 영업일 7일 이내

회사 시스템 점검·국가 공휴일 등의 사유로 처리 기간이 연장될 수 있으며, 그 경우 회원에게 사전 고지합니다.

4.환불이 불가한 경우

  1. 회원이 허위·기재 누락된 자료를 제출하여 환급이 거절된 경우 (이용약관 제8조 위반)
  2. 본인 인증 절차에서 회원의 귀책 사유로 인증이 실패하여 처리가 불가능한 경우
  3. 환급금이 회원 계좌로 정상 입금된 후 단순 변심에 의한 환불 신청 (회사 귀책 제외)
  4. 회원이 본인 명의가 아닌 타인 명의로 신청한 경우
  5. 이미 동일 사유로 환불을 받은 경우

5.법적 권리와의 관계

본 환불정책은 「전자상거래 등에서의 소비자보호에 관한 법률」, 「약관의 규제에 관한 법률」 등 관계 법령에서 규정한 소비자의 권리(청약 철회권 등)를 침해하지 아니하며, 본 정책과 법령이 충돌하는 경우 회원에게 더 유리한 규정이 우선 적용됩니다.

특히 「전자상거래법」 제17조에 따른 청약 철회 기간(통상 7일)은 본 정책의 단계적 환불과 별개로 보장되며, 회사가 디지털콘텐츠로서 별도 고지한 경우 외에는 회원의 청약 철회권이 제한되지 아니합니다.

6.분쟁 해결

환불에 관한 분쟁이 발생할 경우 회사와 회원은 우선 상호 협의를 통해 해결합니다. 협의로 해결되지 않는 경우 회원은 한국소비자원, 전자거래분쟁조정위원회 등을 통한 분쟁 조정을 신청할 수 있습니다.

  • 한국소비자원: 1372 · www.kca.go.kr
  • 전자거래분쟁조정위원회: www.ecmc.or.kr
