# 📚 전자책팀 (Ebook Division)

> **Amazon KDP 기반 글로벌 전자책 자동 집필·출판**
> 🥇 **수익 1순위** — 즉시 시작 가능, 2~4주 내 첫 수익.

## 📍 기본 정보

- **코드명**: `ebook`
- **타겟**: 🌏 글로벌 (영어 1순위)
- **언어**: English (시장 100배 큼)
- **수익 모델**: Amazon KDP 로열티 (35% or 70%)
- **수익 우선순위**: 🥇 **1순위**
- **총괄팀장**: `gm-ebook`

## 🌏 전략

### 왜 영어인가
- 한국 전자책 시장 << 영어 시장 (100배 이상)
- 영어는 KDP, Apple Books, Google Play Books, Kobo 전부 커버
- 심사·서류 불필요 (KDP 즉시 등록)
- 로열티 70% 달성 조건 쉬움 ($2.99~$9.99 가격)

### 첫 타깃: **Low-content books**
- 플래너 / 저널 / 노트북 / 워크북
- 집필 난이도 ↓, 표지·디자인 승부
- 2주 내 첫 책 출간 가능
- 니치 키워드 공략 (ex: "keto meal planner 2026")

### 2차: **Niche non-fiction**
- Self-help (mystic/astrology — 천명당 연계)
- How-to 가이드
- 10k~30k words
- 1개월 집필

## 👥 하위팀 (5팀)

| 팀 | 코드 | 담당 | 단계 |
|---|---|---|---|
| 시장조사팀 | `lead-ebook-research` | Amazon 베스트셀러 분석, 키워드 수요, 경쟁 강도 | 1 |
| 집필팀 | `lead-ebook-writing` | 목차 → 원고, 분량 관리, 예시 | 2 |
| 편집·교정팀 | `lead-ebook-editing` | 문체 교정, 팩트체크, 중복 제거 | 3 |
| 디자인팀 | `lead-ebook-design` | 표지, 내지 레이아웃, EPUB/PDF 변환 | 4 |

### 파이프라인 (순차 실행)
```
시장조사 → 집필 → 편집 → 디자인 → 마케팅 → [CEO 검토] → [CEO 수동 업로드] → 판매
```

## 💰 수익 계산 (예시)

**Low-content book:**
- 가격: $4.99
- 로열티: 35% = $1.75/권
- 월 판매 10권: $17.5 (≈ ₩24,000)
- 월 판매 100권: $175 (≈ ₩245,000)
- 10권 포트폴리오, 권당 월 50권 판매: $875 (≈ ₩1,225,000)

**Niche non-fiction:**
- 가격: $9.99
- 로열티: 70% = $6.99/권
- 월 판매 50권: $349 (≈ ₩489,000)

## ⚠️ 주의사항

1. **AI 생성 표기 의무**: Amazon KDP는 2023년부터 AI 생성 콘텐츠 선언 필수
2. **저작권**: 표지 이미지는 AI 생성 (저작권 free) — 실존 인물·캐릭터 사용 금지
3. **품질 관리**: 리뷰 1~2★ 폭탄 방지 — 최소 수준 이상 보장
4. **포트폴리오 전략**: 1권 대박 노리지 말고 **10권 꾸준히** 접근
