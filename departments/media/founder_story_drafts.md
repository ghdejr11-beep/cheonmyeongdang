## 4. "The 11-field checklist that saved my product launches"

> Last week I forgot to fill the "detailed description" field on a product page. Twice in a row. The user (myself, holding myself accountable) was not pleased.
>
> Now every product launch goes through an 11-field checklist:
> - title / thumbnail / one-liner / price / detailed desc / notice / announcement / group / thanks message / terms checkbox / files
>
> The lesson: the work is never the bottleneck. Memory is. Build the checklist.
