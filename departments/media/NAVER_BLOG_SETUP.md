### 📌 Post 2: KORLENS 런칭 (카테고리: KORLENS)

**제목**: `같은 한국, 다른 관점 | KORLENS가 바꾸는 여행 큐레이션`

**태그**: `#KORLENS #한국관광 #AI여행 #TourAPI #여행큐레이션`

**본문**:
```
https://korlens.vercel.app

## 🔍 KORLENS란?

한국을 17개 광역시도로 나누고, 192개 관광지를
**4가지 관점(외국인·커플·가족·솔로)으로 재해석**하는 AI 여행 큐레이션입니다.

## ✨ 왜 만들었나요?

기존 관광 앱들은 모든 여행자에게 동일한 정보를 보여줍니다.
경복궁을 외국인에게도, 커플에게도, 가족에게도 똑같이 설명하죠.

KORLENS는 다릅니다.
- 🧳 외국인: "영어 해설 투어 11·14시 · 3호선 경복궁역 3분"
- 💑 커플: "경회루 야경 포토스팟 · 삼청동 데이트 코스 연계"
- 👨‍👩‍👧 가족: "수문장 교대식 10·14시 · 유모차 대여 무료"
- 🚶 솔로: "향원정 조용한 산책 루트 · 평일 오전 한산"

같은 장소가 완전히 다른 여행이 됩니다.

## 🎯 핵심 기능

1. **AI 큐레이터 챗봇** — "서울 커플 4시간" 입력 → 실제 장소 추천
2. **혼잡도 필터** — 한산·보통·붐빔 + 유형별 자동 추천
3. **무장애 편의정보** — 열린관광 API 연동
4. **영문 지원** — 외국인 탭에서 한국관광공사 공식 영문 데이터

## 🏆 2026 관광데이터 활용 공모전 출품

한국관광공사 OpenAPI 4종(국문·영문·빅데이터·무장애) 활용.

👉 지금 써보기: https://korlens.vercel.app
```
