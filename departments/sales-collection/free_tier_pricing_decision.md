## 의사결정 프레임 (Sean Ellis 기준)

### Option A — 1회 전체 풀이 무료 (월 1회 한정)
- **장점**: 첫 결제 결정 friction 최저, viral 가능성 ↑
- **단점**: AI 토큰 비용 noise (Claude API $0.05~0.10/풀이)
- **임계 매출**: 월 신규 사용자 1,000명 가정 시 매월 $50 ~ $100 추가 비용

### Option B — 정밀 풀이 첫 1페이지만 무료 preview (현재)
- **장점**: 결제 conversion friction 적당, 비용 0
- **단점**: 사용자 "preview만 보고 이탈" 가능
- **현재 추정 conv**: 1~3% (감)

### Option C — 월회원 첫 7일 무료 (Stripe-style trial)
- **장점**: recurring 수익 ↑, LTV 측정 가능
- **단점**: 갤럭시아/PayPal recurring 셋업 필요. 환불 abuse risk.

## 권장 (memory `revenue_first` 기준)

**Option B 유지 + Option A 의 lite 버전**: "친구 초대 1명 시 1회 무료 정밀 풀이" 추천 코드 시스템.

- 친구 초대 → 친구가 무료 풀이 사용 → 두 명 다 정밀 풀이 1회 무료 체험권 부여
