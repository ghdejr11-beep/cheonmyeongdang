## 回答

韓國的 **궁합 (Gunghap)** 是「**16 字完整對照**」的合婚法,源於朝鮮王朝 (李氏王朝)600 年前的宮廷婚配制度,至今仍是韓國 90% 家庭婚前必做的步驟。它與台港的合八字共用基礎,但**分析層級更深**。

**1. 五層分析法 (16 字並列比對)**

**① 日干關係 (你 vs TA 的本質配對)**
- 天干合 (甲己 / 乙庚 / 丙辛 / 丁壬 / 戊癸) → 自然吸引、互補
- 天干沖 (甲庚 / 乙辛 / 丙壬 / 丁癸 / 戊甲...) → 摩擦、磨合
- 同類干 → 價值觀相似但盲點重疊

**② 五行互補度**
- 你缺的五行 TA 有,TA 缺的你有 → **長期穩定**
- 兩人同元素過旺 → 互相燃燒
- 木火 / 火土 / 土金 / 金水 / 水木 相生組合最佳

**③ 神煞活化**
- **天乙貴人:** TA 成為你人生的貴人/老師
- **桃花:** 強烈肉體吸引力,但需經營深度
- **驛馬:** 適合一起出行/移居,定居反而不順
- **白虎:** 情感激烈,需要成熟度

**④ 隱藏衝突點 (地支沖)**
- 子午沖 → 自由 vs 安定
- 卯酉沖 → 慢工 vs 速效
- 寅申沖 → 行動 vs 計畫
- 巳亥沖 → 熱情 vs 冷靜
- 辰戌沖 → 包容 vs 原則
- 丑未沖 → 守舊 vs 革新

事先知道哪個沖,等於知道**未來吵架的固定議題**。

**⑤ 大運同步度**
- 兩人都在上升大運 → 關係輕鬆
- 一人上升 / 一人下降 → 不平衡感,但**不代表分手**
