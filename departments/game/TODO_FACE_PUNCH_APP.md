# 🥊 얼굴 때리기 앱 (기획 대기 중)

**지시일**: 2026-04-22
**지시자**: CEO (홍덕훈)
**우선순위**: P2 (HexDrop 안정화 이후 착수)

---

## 핵심 컨셉
- 캐릭터 얼굴이 **빈칸**으로 제시됨
- 유저가 **얼굴 사진 업로드** (갤러리/카메라)
- 업로드된 얼굴이 캐릭터 자리에 합성됨
- 유저가 **탭/스와이프**해서 얼굴을 "때리는" 스트레스 해소 게임

## 핵심 매력 (왜 뜰까)
- **개인화 UGC**: 본인·친구·직장상사·전남친/전여친 사진 → 입소문 확산
- **스트레스 해소 니치**: 스트레스 볼 앱류 수천만 다운로드 시장
- **짧은 세션**: 1분 미만, 광고 수익 모델 최적
- **바이럴성**: "이 사람 얼굴 올려놓고 때리기" 짤 → SNS

## 기술 스택 (권장)
- **플랫폼**: Capacitor (천명당/HexDrop과 동일 인프라 재활용)
- **얼굴 인식**: `@vladmandic/face-api` (메모리 기록: justadudewhohacks 버전 호환성 문제, 반드시 vladmandic 포크 사용)
- **애니메이션**: CSS transform + Canvas 또는 Lottie
- **저장**: 로컬 only (얼굴 이미지 서버 업로드 X → 개인정보 리스크 0)
- **광고**: AdMob (쿤스튜디오 전용 ID 발급 필수 — 천명당 AdMob 이슈와 별개 ID)

## 수익 모델
- 배너 광고 (하단)
- 리워드 광고: "3번 때리면 다음 스페셜 효과"
- 인앱결제: 스페셜 효과 팩 (폭발/번개/파이)
- 없음(무료 + 광고만) 로 시작

## 개인정보·법률 리스크 ⚠️
- 얼굴 사진 업로드 + 폭력적 행위 = 실제 인물 **초상권 침해** 가능
- 대응:
  1. 약관에 "본인 또는 동의받은 사람만 등록" 명시
  2. 업로드 이미지 서버 저장 절대 금지 (로컬만)
  3. 스토어 등록 시 "Cartoon violence, no real people" 가이드라인 체크
  4. 연령 등급: 12+ 이상
- **메모리 기록 검색 필수**: "얼굴 이미지 수집·저장하는 앱은 거부 (facial images 가이드라인)" — Claude가 얼굴 이미지 분석·수집은 거절하므로 **클라이언트 only** 로직으로 설계

## 시장 경쟁사
- Stress Ball 계열 (Candy Bang, Slap King) — 얼굴 UGC 없음
- Slap Kings (MOTION VOLT) — 다운로드 50M+, but 고정 캐릭터
- **틈새**: "너 얼굴 업로드" + "친근한 펀치 게임" 조합 앱 미포화

## MVP 단계 (4주)
- Week 1: Capacitor 프로젝트 셋업 + 얼굴 업로드/크롭 UI
- Week 2: face-api로 눈/코/입 landmark 추출 → 캐릭터 얼굴 자리 매핑
- Week 3: 펀치 탭 인터랙션 + 데미지 애니메이션 (빨간/파란 번짐)
- Week 4: 사운드 + 점수 + 공유 기능 + 광고 붙이기
