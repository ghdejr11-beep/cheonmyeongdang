## 🗺️ 사업 방향

### 1. 앱: 한국 여행 가이드
- **타깃**: 20~30대 국내 여행자 + 인바운드 관광객
- **차별화**: AI 큐레이션 코스 추천 ("당일치기 부산 맛집", "제주 3박4일 힐링")
- **수익**: 광고 + 프리미엄 코스 IAP + 제휴 커미션

### 2. B2B: 지도 API 판매
- **타깃**: 여행사, 관광 스타트업, 지자체
- **제공**: 한국 관광지 DB + POI (Point of Interest) + AI 추천 API
- **수익**: 월 구독 or 호출당 과금
- **경쟁**: Google Maps Platform, 네이버 지도 API (비싸고 범용)

### 차별화 포인트
- **한국 여행 특화**: 관광지·카페·맛집 DB 큐레이션
- **AI 추천**: 단순 POI 아닌 "코스" 제공
- **저렴한 B2B 가격**: Google 대비 저렴
