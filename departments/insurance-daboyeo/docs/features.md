## 2️⃣ 내보험다보여 API 연동 설계

**상태:** 설계 완료, 심사 대기.

- **URL**: `fine.fss.or.kr`

- **필요 제출**: 사업자등록증 + 개인정보 처리방침 URL
- **심사 기간**: 3~5일

## 3️⃣ 마이헬스웨이 API 연동 설계

**상태:** 설계 완료, 심사 대기.

- **기관**: 보건복지부
- **URL**: `myhealthway.go.kr`
- **목적**: 심평원 **5년 진료기록** 조회
- **인증 체인** (5단계 진행 바):
  1. 카카오/PASS/공동인증서 본인 인증
  2. 건강보험공단 연동
  3. 심평원 동의
  4. 데이터 조회
  5. 앱 표시
- **심사 기간**: 5~10일
