# 보험 API 심사 신청 체크리스트

## 1. 내보험다보여 API (금융감독원)
- **사이트**: https://fine.fss.or.kr
- **심사 기간**: 3~5일
- **필요 서류**:
  - ✅ 사업자등록증 (552-59-00848, 쿤스튜디오)
  - ✅ 개인정보처리방침 URL: https://ghdejr11-beep.github.io/cheonmyeongdang/privacy.html
  - ✅ 이용약관 URL: https://ghdejr11-beep.github.io/cheonmyeongdang/terms.html
  - ❓ 서비스 소개서 (앱 스크린샷 + 기능 설명)
  - ❓ 보안 서약서 (금감원 양식)

## 2. 마이헬스웨이 API (보건복지부)
- **사이트**: https://myhealthway.go.kr
- **심사 기간**: 5~10일
- **필요 서류**:
  - ✅ 사업자등록증
  - ✅ 개인정보처리방침 URL
  - ❓ 앱 서비스 화면 캡처
  - ❓ 데이터 활용 계획서

## 신청 순서
1. 금감원 fine.fss.or.kr 접속 → API 신청
2. 사업자등록증 PDF 업로드
3. 개인정보처리방침 URL 입력
4. 서비스 설명 작성
5. 심사 대기 (3~5일)

※ 두 곳 동시에 신청 가능. 기다리는 시간이니 빨리 넣을수록 좋음.
