什麼是韓國八字算命？先從「사주팔자」說起

如果你看過韓劇，一定聽過劇中人物說要去「看사주」——這就是韓國的四柱八字命理，也是東亞漢字文化圈共同的命理傳統。

「사주팔자（四柱八字）」字面意思就是「四根柱子、八個字」。所謂四柱，是指一個人出生的年、月、日、時，而每一柱都由一個天干與一個地支組合而成，四柱加起來就有八個字，這就是「八字」名稱的由來。

在台灣、香港、中國也有相同的命理體系，但韓國版本在文化融合過程中發展出了自己的特色，甚至衍生出獨特的術語與流派。許多韓國人在結婚、求職、搬家前都會去找命理師「看사주」，這個傳統至今仍然相當普遍。

韓國八字的歷史背景：從宮廷到現代街頭

朝鮮時代的命理文化

八字命理傳入韓半島的時間可以追溯到統一新羅時代（西元7世紀左右），隨著漢字文化與道教、儒學一同傳入。到了朝鮮王朝（1392–1897），四柱命理更被納入官方體制——朝廷設有專門的「觀象監」，負責天文、曆法與占卜事務，命理師的地位不亞於現代的專業顧問。

皇室貴族在選擇繼承人、決定婚配、選定吉日時，都會參考八字與風水。這種傳統一代一代傳承下來，逐漸從宮廷普及到民間。

現代韓國的算命文化

今天走在首爾的鐘路（종로）或弘大（홍대）附近，你還是能看到掛著「사주·타로」招牌的命理小店。根據韓國媒體統計，每逢年初、大學學測（수능）前後，以及情人節、新年等時節，命理諮詢的需求量都會顯著上升。

有趣的是，不少韓國年輕世代（甚至是KPOP粉絲）也開始對八字產生興趣，把它視為一種「了解自己性格與命運」的工具，就像西方星座或MBTI一樣流行。

八字的基本架構：天干與地支

要看懂八字，首先需要認識兩個核心要素：天干與地支。

十天干

天干共有十個，代表陰陽與五行的組合：

| 天干 | 韓語 | 五行 | 陰陽 | |------|------|------|------| | 甲 | 갑 | 木 | 陽 | | 乙 | 을 | 木 | 陰 | | 丙 | 병 | 火 | 陽 | | 丁 | 정 | 火 | 陰 | | 戊 | 무 | 土 | 陽 | | 己 | 기 | 土 | 陰 | | 庚 | 경 | 金 | 陽 | | 辛 | 신 | 金 | 陰 | | 壬 | 임 | 水 | 陽 | | 癸 | 계 | 水 | 陰 |

十二地支

地支共有十二個，對應十二生肖與不同的時辰：

| 地支 | 韓語 | 生肖 | 時辰（舊制） | |------|------|------|----------| | 子 | 자 | 鼠 | 23:00–01:00 | | 丑 | 축 | 牛 | 01:00–03:00 | | 寅 | 인 | 虎 | 03:00–05:00 | | 卯 | 묘 | 兔 | 05:00–07:00 | | 辰 | 진 | 龍 | 07:00–09:00 | | 巳 | 사 | 蛇 | 09:00–11:00 | | 午 | 오 | 馬 | 11:00–13:00 | | 未 | 미 | 羊 | 13:00–15:00 | | 申 | 신 | 猴 | 15:00–17:00 | | 酉 | 유 | 雞 | 17:00–19:00 | | 戌 | 술 | 狗 | 19:00–21:00 | | 亥 | 해 | 豬 | 21:00–23:00 |

天干與地支兩兩配對，60年為一個完整循環，也就是我們常說的「六十甲子」。

如何排出自己的八字？

排八字需要你的出生年、月、日、時（越精確越好），利用萬年曆換算出對應的天干地支。以下是基本步驟：

1. 確認出生年份的天干地支：例如1995年是乙亥年 2. 查出生月的月柱：依照農曆月份與節氣來定 3. 查出生日的日柱：直接查萬年曆對應的干支日 4. 查出生時辰的時柱：根據出生時間換算對應的地支，再搭配日干推算時干

初學者建議使用免費的八字排盤工具（網路上很多）來自動生成命盤，再逐步學習每個柱的含義。

> 小提醒：韓國命理師特別重視「日主」，也就是日柱天干——這代表你本人的核心能量與性格特質，是解讀整個命盤的起點。

五行與十神：八字解讀的核心邏輯

五行相生相剋

五行（木、火、土、金、水）是理解八字的關鍵。它們之間有兩種基本關係：

  • 相生（促進）：木生火 → 火生土 → 土生金 → 金生水 → 水生木
  • 相剋（制約）：木剋土 → 土剋水 → 水剋火 → 火剋金 → 金剋木
命盤中五行的平衡狀態，決定了一個人的個性傾向與運勢走向。五行過旺或過弱，都可能帶來失衡的狀況。

十神（십신）：韓國命理的重要分析工具

十神是以「日主」為基準，將命盤中其他七個字與日主的五行生剋關係，分類成十種角色，分別對應人生中不同的面向：

| 十神 | 代表意涵 | |------|----------| | 比肩、劫財 | 兄弟姐妹、競爭者、自我意志 | | 食神、傷官 | 才華、表達、子女（女命） | | 正財、偏財 | 財富、父親、工作收入 | | 正官、七殺 | 官職、丈夫（女命）、紀律 | | 正印、偏印 | 母親、學業、貴人 |

韓國命理師非常善用十神來分析感情、事業與家庭關係，這也是韓國八字與其他流派較為顯著的特色之一。

韓國八字與台灣八字有什麼不同？

很多台灣朋友會好奇：韓國八字和我們平常接觸的八字有差別嗎？

基本理論架構是相同的，但有幾點值得注意：

  • 流派差異：韓國命理發展出「자평명리（子平命理）」與「격국론（格局論）」等本土化流派，部分術語與台灣通用命理書略有不同。
  • 用語不同：即使概念相同，韓語命名有時與漢字表述有微妙差異，閱讀韓文命理資料時需要適應。
  • 文化應用重點：韓國人特別重視用八字來分析「합（合）」，也就是兩人八字是否相合，這在結婚前尤其普遍，稱為「궁합（宮合）」。
  • 大運計算方式：大運（10年一輪的運勢週期）的起運時間計算方式與中台地區主流算法基本一致，但部分韓國流派有細微調整。

初學者最常見的疑問解答

Q1：不知道出生時辰，還能算八字嗎？

可以，但命盤的精準度會降低。沒有時柱的情況下，命理師通常只能解讀年、月、日三柱，也就是六個字。建議向家人確認或查閱出生記錄。

Q2：農曆和國曆，要用哪個算？

排八字以國曆（西曆）出生日期為準，再透過萬年曆換算農曆干支。特別要注意節氣的交界——月柱的換算是以節氣而非農曆初一為分界點。

Q3：韓國算命師都會說韓語，語言不通怎麼辦？

現在有越來越多韓國命理App與網站提供中文或英文介面，可以先用工具排出自己的命盤，再對照說明自學基礎知識。如果有機會去首爾，鐘路三街一帶有部分命理師有中文服務能力。

Q4：八字算命是迷信嗎？

這個問題沒有標準答案。從學術角度來看，命理學是東亞傳統哲學的一部分，融合了天文曆法、陰陽五行思想與心理洞察。許多人把它當作認識自我的工具，而非絕對的預言。理性參考、不過度依賴，是最健康的態度。

如何開始自學韓國八字？推薦學習路徑

如果你對韓國八字有興趣，以下是一個循序漸進的自學路線：

第一階段：打好基礎

  • 熟記十天干與十二地支
  • 理解五行相生相剋關係
  • 學會用萬年曆排出自己的命盤
第二階段：理解結構
  • 學習十神的含義與對應關係
  • 了解「日主強弱」的基本判斷方法
  • 認識大運與流年的概念
第三階段：深入應用
  • 研究格局（格局論）與用神選取
  • 學習宮合（궁합）的基本判斷
  • 閱讀韓國命理師著作或觀看韓語命理教學影片
有一位朋友曾經分享她的經驗：原本只是好奇去韓國旅遊時順道「看了個사주」，回台後卻被命理師提到的某些性格特質深深打動，從此開始認真研究四柱命理。她說最大的收穫不是「預測未來」，而是透過命盤重新認識了自己的優勢與盲點——這或許就是八字命理歷久不衰的真正魅力。

結語：八字是認識自己的一面鏡子

韓國八字算命並不神秘，它是一套有系統的傳統哲學工具，幫助人們從出生時間的宇宙座標中，找尋自身與環境的對應關係。無論你是想深入鑽研命理學，還是純粹出於對韓國文化的好奇，從了解天干地支、五行原理開始，是最扎實的入門路徑。

命理的智慧在於「知命而不認命」——透過了解自己的特質與潛在趨勢，做出更有意識的選擇，這才是四柱八字真正的價值所在。
