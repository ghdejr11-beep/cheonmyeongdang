# AI 종합소득세 신고 — 30분에 끝내는 방법 (2026)

종합소득세 신고에 평균 **5시간**이 걸립니다 (자영업자 600만 명 기준). 5월 한 달 동안 600만 명 × 5시간 = 3,000만 시간이 신고에 사용됩니다. 시간당 1만 원만 잡아도 **3,000억 원의 시간 비용**이 매년 사라집니다.

세금N혜택은 Claude AI + 국세청 API 정식 연동으로 이 시간을 **30분**으로 단축했습니다.

| 단계 | 직접 신고 (홈택스) | 세무사 의뢰 | **AI 신고 (세금N혜택)** |
|---|---|---|---|
| 자료 정리 | 2~3시간 | 1시간 (자료 전달) | **5분** (사진/PDF 업로드) |

## AI 신고가 빠른 이유 4가지

### 1. 국세청 API 정식 연동
홈택스 로그인 후 **사용자 동의 1회**로 국세청 데이터(소득·지급명세서·연말정산 자료)를 직접 가져옵니다. 종이로 모으거나 캡처할 필요가 없습니다.

### 2. AI가 누락 항목 자동 점검
- 노란우산공제 가입 여부
- 연금저축/IRP 한도 잔여
- 신용카드 사업 사용분 안분
- 감가상각 자산 잔액

위 항목을 AI가 자료에서 찾아 빠진 부분을 **추천**합니다.

### 3. 자연어 대화로 입력
"작년에 카메라 200만 원짜리 샀는데 어떻게 처리해요?"
→ AI가 답변 + 자동으로 감가상각 항목에 등록.

복잡한 세법 용어를 모르고도 진행 가능합니다.

### 4. 세무사 검수 (옵션)
AI 결과가 못 미더우면 **+5만 원으로 세무사 검수**를 추가할 수 있습니다 (단순 케이스 한정).

## 30분 신고 — 실제 단계

### Step 1 (5분) — 가입 + 자료 업로드
- [tax-n-benefit-api.vercel.app](https://tax-n-benefit-api.vercel.app) 접속
- 카카오/네이버 로그인 (1초)
- 사업자등록증/사업자번호 입력
- (선택) 카드명세서 PDF·세금계산서 업로드

### Step 2 (5분) — 국세청 자동 연동
- 홈택스 인증 (공동인증서 또는 카카오 인증)
- 사용자 동의 1회 → 자동 데이터 가져오기
- 소득·지급명세서·연말정산자료 자동 입력

### Step 3 (10분) — AI 대화로 누락 항목 점검
- AI가 7~10가지 추가 질문
  - "노란우산공제 가입하셨나요?"
  - "월세 사시나요? 시가표준액 4억 이하인가요?"
  - "사업용으로 쓰는 차량/노트북 있나요?"
- 답변할 때마다 환급액 실시간 업데이트

### Step 4 (5분) — 결과 요약 + 세무사 검수 옵션
- 환급액 / 세액 / 누락 가능 항목 요약
- (옵션) +5만 원 세무사 검수 추가
- 신고 동의 체크

### Step 5 (5분) — 제출 + 결제 약속
- 홈택스에 1클릭 제출
- 환급 입금 후 9.9% 자동 차감 (선불 X)
- 환급 0원이면 수수료 0원 (환불보장 정책 적용 시)
