AI 신고가 빠른 이유 4가지

1. 국세청 API 정식 연동

홈택스 로그인 후 사용자 동의 1회로 국세청 데이터(소득·지급명세서·연말정산 자료)를 직접 가져옵니다. 종이로 모으거나 캡처할 필요가 없습니다.

2. AI가 누락 항목 자동 점검

  • 노란우산공제 가입 여부
  • 연금저축/IRP 한도 잔여
  • 신용카드 사업 사용분 안분
  • 감가상각 자산 잔액

3. 자연어 대화로 입력

"작년에 카메라 200만 원짜리 샀는데 어떻게 처리해요?"
→ AI가 답변 + 자동으로 감가상각 항목에 등록.

4. 세무사 검수 (옵션)

AI 결과가 못 미더우면 +5만 원으로 세무사 검수를 추가할 수 있습니다 (단순 케이스 한정).

30분 신고 — 실제 단계

Step 1 (5분) — 가입 + 자료 업로드
  • tax-n-benefit-api.vercel.app 접속
  • 카카오/네이버 로그인 (1초)
  • 사업자등록증/사업자번호 입력
  • (선택) 카드명세서 PDF·세금계산서 업로드
Step 2 (5분) — 국세청 자동 연동
  • 홈택스 인증 (공동인증서 또는 카카오 인증)
  • 사용자 동의 1회 → 자동 데이터 가져오기
  • 소득·지급명세서·연말정산자료 자동 입력
Step 3 (10분) — AI 대화로 누락 항목 점검
  • "노란우산공제 가입하셨나요?"
  • "월세 사시나요? 시가표준액 4억 이하인가요?"
  • "사업용으로 쓰는 차량/노트북 있나요?"
  • 답변할 때마다 환급액 실시간 업데이트
Step 4 (5분) — 결과 요약 + 세무사 검수 옵션
  • 환급액 / 세액 / 누락 가능 항목 요약
  • (옵션) +5만 원 세무사 검수 추가
  • 신고 동의 체크
Step 5 (5분) — 제출 + 결제 약속
  • 홈택스에 1클릭 제출
  • 환급 입금 후 9.9% 자동 차감 (선불 X)
  • 환급 0원이면 수수료 0원 (환불보장 정책 적용 시)

자주 묻는 질문

Q. AI가 잘못 입력하면 책임은 누가 지나요?
A. AI 분석 + 세무사 검수 옵션을 거친 신고 건은 가산세 책임 보장 정책 적용. 단, 사용자가 자료를 누락 제출한 경우 제외.

Q. 데이터 보안이 걱정됩니다.
A. 국세청 API 정식 연동(MyData 형식) + 신고 후 자료 즉시 폐기 + HTTPS 전구간 암호화. 빌게이트 PG는 한국 PCI-DSS 인증.

Q. 환급 입금이 안 되면 9.9%는 어떻게 되나요?
A. 환급이 0원이면 9.9%는 청구되지 않습니다 (환불보장 정책 적용 시).
