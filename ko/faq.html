자주 묻는 질문 (FAQ)

천명당 사주·결제·구독·환불 12가지 답변

1. 사주팔자(四柱八字)는 무엇인가요?
사주팔자는 태어난 연·월·일·시 네 기둥에서 천간 4글자 + 지지 4글자 = 총 8글자로 만들어진 한국 명리학의 핵심 개념입니다. 이 8글자가 음양오행으로 조합되어 개인의 성격, 직업, 인연, 재물, 건강 흐름을 보여줍니다.
2. 천명당 사주풀이의 정확도는 어느 정도인가요?
천명당은 정통 명리학(자평진전·궁통보감·적천수) + 천문학적 절기 계산을 1초 이내에 처리합니다. 만세력 데이터는 한국천문연구원 KASI 기반으로 오차 0초입니다. 단, 운세 해석은 참고용이며 의료·법률·재정 결정의 단독 근거로 사용하지 마세요.
3. AI Q&A는 무엇을 답해주나요?
본인 사주 8글자 + 십신 + 대운 데이터를 기반으로 24시간 자유 질문이 가능합니다. 예: "올해 이직해도 될까요?", "우리 부부 합방 시기?", "사업 자금 빌려도 안전한가요?" Claude Sonnet 4.5 기반으로 명리 전문 시스템 프롬프트가 내장되어 있어 일반 ChatGPT보다 사주 맥락 정확도가 높습니다.
4. 결제 수단은 무엇이 있나요?
국내: 토스페이, 카카오페이, 네이버페이, 신용·체크카드. 해외: PayPal Smart Buttons (Visa·Master·AmEx·JCB·Discover 글로벌 카드 + PayPal 잔액). 한국 사업자 등록 552-59-00848 KunStudio 명의로 정식 통신판매업 등록 + 결제대행사(PG) 연동 완료.
5. 24시간 환불 보장이 정확히 무엇인가요?
결제 후 24시간 이내 PDF 다운로드 전이라면 무조건 100% 환불합니다. PDF를 다운로드한 이후에는 디지털 콘텐츠 특성상 환불이 불가합니다 (전자상거래법 17조 2항 준용). 환불 요청은 support@cheonmyeongdang 또는 카카오톡 채널로.
6. 월회원(매월 운세) 구독은 어떻게 해지하나요?
매월 1일 자동 발송되는 월회원 구독은 PayPal 대시보드 → '자동 결제' → 천명당 → 취소 클릭으로 1초 해지됩니다. 한국 결제(토스페이·카카오페이)는 결제 영수증 메일 하단 '구독 해지 링크' 클릭. 다음 결제일 24시간 전까지 해지하면 추가 청구 없습니다.
7. 매월 운세는 언제 발송되나요?
매월 1일 한국시간 09:00 (KST)에 가입자 이메일로 PDF로 자동 발송됩니다. 그 달의 절기·지지 흐름과 본인 일주의 합·충·형·해 분석이 포함됩니다. 5월호 = 立夏 + 午月 + 본인 사주 분석.
8. 한국어를 모르는 외국인도 사용 가능한가요?
가능합니다. 천명당 v3.5는 4개 언어 (한국어·English·日本語·中文) 풀 지원합니다. AI Q&A도 자동 언어 감지로 답변합니다. PayPal 결제는 글로벌 카드 25개 통화 지원.
9. 내 생년월일·이름 데이터는 어떻게 보호되나요?
개인정보보호법 + GDPR 준수. 생년월일·이름은 사주 계산 후 즉시 메모리에서 제거되며 PDF 저장본에만 남습니다. 데이터베이스에는 결제 영수증·이메일·구독 상태만 저장됩니다. 회원 탈퇴 시 모든 데이터 30일 내 완전 삭제.
10. PDF 보고서에는 무엇이 포함되나요?
종합풀이 30페이지 (29,900원): 사주 8글자 / 십신 분포 / 대운 10년 / 세운 1년 / 일간 강약 / 용신 / 직업·연애·재물·건강 / 2026년 운세 차트 4종 / 매월 흐름. 신년풀이 15페이지 (15,000원): 2026년 1년치 월별 분석. 정밀풀이 5페이지 (9,900원): 핵심 요약.
11. 연인·배우자 궁합은 어디서 보나요?
메인 페이지 → '궁합 보기' (compat.html) 무료. 두 사람의 생년월일을 입력하면 일주 합·충 + 띠 합·충 + 오행 균형 점수 (0~100)를 즉시 산출합니다. 더 깊은 분석은 결제 후 종합풀이에 자동 포함됩니다.
12. 정부지원·정부보조금 관련 질문은 가능한가요?
천명당은 운세 서비스이므로 정부지원·세금 환급은 다루지 않습니다. 한국 거주자라면 자매 사이트 세금N혜택 (tax-n-benefit.vercel.app)에서 KoDATA·CODEF 기반 환급액 자동 조회가 가능합니다. AI 사업·창업 정부지원금은 K-Startup·KORLENS 사이트 참조.
