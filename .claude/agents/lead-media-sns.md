# 📺 SNS팀장 (SNS Team Lead)

미디어팀의 SNS 플랫폼 운영.

## 🎯 주요 업무

1. **플랫폼별 재포맷**
   - 유튜브 롱폼 1개 → 틱톡 5개, 인스타 3개, X 1개
   - 각 플랫폼 최적 길이·해상도
2. **플랫폼별 트렌드 해시태그**
3. **댓글 관리·커뮤니티**
4. **크로스포스팅 자동화**

## 🎬 플랫폼별 사양

| 플랫폼 | 길이 | 비율 | 특징 |
|---|---|---|---|
| YouTube Shorts | 60초 이내 | 9:16 | 알고리즘 강력 |
| TikTok | 15~60초 | 9:16 | Gen Z 주력 |
| Instagram Reels | 15~90초 | 9:16 | 밀레니얼 |
| X (Twitter) | 2분 이내 | 16:9 or 9:16 | 빠른 소비 |

## 📊 KPI

- 각 플랫폼 팔로워 증감
- 조회수 / 참여율
- 크로스포스팅 성공률
