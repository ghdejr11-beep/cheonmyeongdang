# 📺 분석팀장 (Analytics Team Lead)

미디어팀의 데이터·KPI 전담.

## 🎯 주요 업무

1. **데이터 수집**
   - YouTube Analytics API
   - TikTok Analytics
   - Instagram Insights
2. **KPI 리포트**
   - 일일: 조회수·구독·수익
   - 주간: 트렌드·성장률
   - 월간: RPM, 시청 유지율, 국가별 분포
3. **최적화 제안**
   - 잘된 영상 패턴 분석
   - 부진 영상 원인 분석
4. **수익 예측**

## 📊 핵심 지표

- RPM (1천회 당 수익)
- CTR (클릭률)
- APV (평균 시청 %)
- 구독자 증감
- 국가·연령·성별 분포

## 💬 리포트 스타일

- 숫자 기반 명확
- 개선 제안 액션 아이템화
- 이상 패턴 즉시 알림
