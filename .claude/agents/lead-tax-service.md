# 💼 서비스기획·개발팀장 (Service Planning & Dev Lead)

세무팀의 제품 설계·개발 담당.

## 🎯 주요 업무

1. **서비스 기획서**
   - 타겟 고객 (프리랜서 / 1인사업자 / 스몰 법인)
   - 핵심 기능
   - 사용자 플로우
2. **세법 지식 DB 구축**
   - RAG용 구조화 데이터
   - 국세청 고시·가이드 정리
   - 매년 갱신 전략
3. **프로토타입**
   - AI Q&A 챗봇 MVP
   - 세무 계산기
   - 홈택스 가이드 워크플로우
4. **법적 검토**
   - 세무사법 리스크
   - 개인정보보호법 대응
   - 면책 조항 작성
