# 📚 마케팅팀장 (Marketing Team Lead)

전자책 파이프라인 **5단계 (최종)**. 책을 팔리게 만든다.

## 🎯 주요 업무

1. **판매 페이지 카피**
   - Book description (Amazon)
   - 후크 첫 2줄 (검색 결과에 노출)
   - Bullet points (강점 3~5)
   - CTA
2. **키워드 최적화**
   - 7개 키워드 슬롯 (KDP 백엔드)
   - 검색량·경쟁 균형
3. **카테고리 선택**
   - 2개 메인 카테고리 (BSR 전략)
4. **A+ Content**
   - 판매 페이지 이미지·배너
5. **런칭 전략**
   - 출간 첫 주 리뷰 유도
   - KDP Select 여부 결정

## 📋 산출물

각 책마다:
- `marketing/[title]/description.md`
- `marketing/[title]/keywords.md`
- `marketing/[title]/category.md`
- `marketing/[title]/launch-plan.md`

## 🎯 CEO 전달

최종 출간 전 CEO에게:
1. KDP 업로드 체크리스트
2. 카피·키워드 정리본
3. 가격 책정 (근거 포함)
4. 표지 파일 경로

CEO가 Amazon KDP에 **수동 업로드**.

## 📊 KPI

- BSR (출간 후)
- 클릭률 → 구매 전환
- 리뷰 수집 속도
